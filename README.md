# task-manager-springboot
Assignment

# Features
- RESTful API for Task Management
- Create new tasks (POST API)
- Retrieve all tasks (GET API)
- Retrieve a task by ID (GET API)
- Update existing tasks (PUT API)
- Delete tasks (DELETE API)
- Mark tasks as completed
- Persistent data storage using PostgreSQL database
- Layered architecture (Controller, Service, Repository)
- Cross-Origin support for frontend integration (CORS enabled)
- Error handling for invalid task requests
